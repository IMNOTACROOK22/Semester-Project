# Semester-Project
# Develop an online shopping site that can perform the following

  • Must be database driven
  
    o You may choose any dB
    
  • Create/Modify User Information
    o Allow users to register for accounts
  • Create/Modify Items for Sale
    o Must include images
    o Must include price
    o Must include quantity available
    o Must allow ability to add new items
  • Create/Modify Shopping Cart
    o Must show items in cart
    o Must calculate taxes: assume 8.25% tax rate
    o Must allow for discount codes
    o Must show summary of order and allow for order to be placed
  • Ability to search based on criteria:
    o Item/Description
    o Sort by Price
    o Sort by Availability
  • Administrative back end
    o Allow to modify all items
    o Allow for creation of discount codes
    o Allow for creation of sales items
    o Allow to modify users
    o Show currently placed orders
    o Show history of orders
      ▪ Sort by order date
      ▪ Sort by customer
      ▪ Sort by order size in dollar amount
  • You must have an intuitive UI/UX
